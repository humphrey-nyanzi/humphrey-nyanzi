## Projects

### 🏃 High-Speed Running Decay in a Domestic African League
*GPS-based physical performance analysis | 2024/25 season*

Quantifies second-half HSR decay across 16 professional clubs using 3,441 
player-match observations. Positional breakdown, club-level variation (3.8x 
range between highest and lowest decay clubs), and full statistical testing 
(paired t-test, Wilcoxon, ANOVA + Tukey HSD).

**Key finding:** Players average 24.9% decline in high-speed running rate 
from first to second half. Forwards decay fastest (23.9%), defenders slowest 
(12.8%).

[View project →](https://github.com/humphrey-nyanzi/UPL-HSR-Decay)

### ⏱️ Uganda Premier League Goal Timing Analysis
*Six seasons. 3,222 goals. One finding that contradicts what most football research says.*

Longitudinal analysis of when goals are scored in the UPL across 2019/20 
through 2024/25. Documents a halftime-restart vulnerability pattern that 
differs meaningfully from European football research.

**Key finding:** The 46–60 minute window is the most dangerous period in a 
UPL match (17.9% of all goals) — not the final 15 minutes as European 
research consistently shows.

[View project →](https://github.com/humphrey-nyanzi/UPL-goal-timing)
